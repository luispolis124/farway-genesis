# farway-genesis
Explore o Farway Genesis, uma expansão épica para o seu Minecraft! Descubra a poderosa Adiamanta, um minério místico superior ao Netherite. Crie ferramentas lendárias, utilize o Cinzel avançado para transformar blocos e prepare-se para enfrentar criaturas místicas em sua jornada. Um universo de novas possibilidades espera por você. Baixe agora!
